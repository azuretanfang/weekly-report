# HR行业资讯周报 📋

> 聚焦互联网与游戏行业 · HRBP 团队内部参阅

## 项目概述

自动化生成 HR 领域行业资讯周报，覆盖互联网和游戏行业的 HR 热点动态，帮助 HRBP 团队高效掌握行业趋势。

### 核心特色
- 🎯 **聚焦行业**：专注互联网 + 游戏行业 HR 领域
- 🏢 **"与我们相关"标注**：自动识别并高亮与公司直接相关的资讯
- 📊 **五大板块结构化**：政策法规 / 薪酬市场 / 标杆实践 / AI+HR / 热点事件
- 📲 **企微群推送**：通过企业微信群机器人一键分发

## 快速开始

### 1. 生成周报

**方式一：AI 对话生成（推荐）**

在 CodeBuddy 中对话：
> "帮我生成本周的 HR 行业资讯周报，范围 04.07-04.13"

AI 会自动搜索最新资讯 → 结构化分类 → 生成 HTML 文件。

**方式二：从数据文件生成**

```bash
# 准备 JSON 数据文件
python generate_hr_weekly.py --data news_data.json --start 0401 --end 0406

# 查看搜索关键词库
python generate_hr_weekly.py --dry-run
```

数据文件格式：
```json
[
  {
    "category": "政策法规",
    "title": "资讯标题",
    "summary": "详细摘要内容...",
    "impact": "对我们的影响分析（可选，有则自动标注'与我们相关'）",
    "source": "来源"
  }
]
```

### 2. 预览周报

直接用浏览器打开 HTML 文件即可预览：
```bash
open hr-weekly-report-0401-0406.html
```

### 3. 推送到企微群

**首次配置：**
1. 在企业微信群中添加群机器人
2. 复制 Webhook URL
3. 修改 `send_wechat.py` 中的 `WECHAT_WEBHOOK_URL`

**推送命令：**
```bash
# 自动检测最新周报并推送 Markdown 摘要
python send_wechat.py

# 测试模式（不实际发送）
python send_wechat.py --test

# 指定文件并发送图文卡片
python send_wechat.py --file hr-weekly-report-0401-0406 --mode news

# 推送全部格式（摘要 + 卡片 + 长图）
python send_wechat.py --mode all --image hr-weekly-report-0401-0406-hd.png
```

## 每周工作流

```
周一上午
  ↓
AI 搜索上周 HR 资讯（自动化）
  ↓
筛选分类 + "与我们相关" 标注（AI + 人工复审 15min）
  ↓
生成 HTML 周报
  ↓
截图高清长图
  ↓
企微群推送 ✅
```

**预计每周耗时：AI生成 5 分钟 + 人工复审 15-30 分钟 = 30 分钟内**
