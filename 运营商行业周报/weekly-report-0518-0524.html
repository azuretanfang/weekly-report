📈 进展更新：从"战略共识"到"全国上线"，Token 经营进入商业化兑现期

🔄 进展
较上期变化：0501-0510 周报曾以"战略共识"角度报道"从流量到 Token"的行业转向。本周（5/17-5/20）三大运营商集中将其转化为面向 C 端用户的商业化套餐，标志着 Token 经济从战略口号进入资费上线、用户开通的执行阶段。（来源：新华社、腾讯科技 2026/05/20）
🎯 对比
三家 Token 套餐落地节奏：
• 中国电信：5月17日全国试商用，个人/家庭三档（9.9 元/1000万 Tokens、29.9 元/4000万、49.9 元/8000万），开发者及中小微企业另设 39.9-299.9 元档位（来源：腾讯科技 2026/05/20）
• 中国移动：5月20日全国上线"词元(Token)套餐"，最低 5 元月包，统一不同模型量纲为标准积分，与移动云电脑、云手机融合，可在腾讯 WorkBuddy 等主流 AI 应用通用（来源：新华社 2026/05/20）
• 中国联通：四川、上海、湖北等地推出 Token Plan 个人版/团队版，三档可选 600/1200/1800 万 Tokens，价格 7.5-359 元/月，与千兆宽带、AI 云桌面捆绑（来源：腾讯网 2026/05/20）
💡 洞察
三大共性 +一大差异：共性体现在①将 Token 类比"流量包"做规模化普惠；②与现有产品（云电脑、千兆宽带、定向流量）深度捆绑；③切入"统一计量"叙事以化解多模型计费混乱。差异点在于中国移动率先打通"话费支付"，复用其规模优势，最有机会把 Token 变成像短信、流量一样的国民级数字消费品。（来源：媒体综合分析 2026/05/20）

👔 人事变动

省分及子公司密集调整

📋 中移铁通 + 中国电信省分公司密集调整（5/16-5/19）

🆕 子公司
中移铁通：5月19日前后多位省分公司领导调动，涉及子公司治理结构和市场经营条线的人员补位。中移铁通目前由孙卫东任党委书记、董事长，本轮调整聚焦于省分一级。（来源：通信资讯·WXRedian 2026/05/19）
🔄 电信
中国电信：本周公告显示多位部门和子分公司领导调动，是继 4 月份董事会层面调整后又一轮中层级别的换防（来源：通信资讯·WXRedian 2026/05/19）。结合 5月16日"常话短说"汇总，铁塔尹文凯升任党委副书记，运营商整体处在"新班子+新打法"过渡期。

🚀 行业动态

"双万兆"+ Token + 智算并行推进
📡 5G-A/6G

5/17 武汉世界电信日大会：中国移动定调"双千兆向双万兆演进"，加强 5G-A、6G、卫星互联网核心技术攻关

2026 年世界电信和信息社会日大会在武汉召开，中国移动副总经理陈怀达表示，面向"十五五"，中国移动将焕新移动通信、宽带网络、蜂窝物联、卫星互联等业态，推动"双千兆"网络向"双万兆"演进，加快建设通感算智融合、空天地海一体的下一代信息基础设施。（来源：腾讯网"数智早参"2026/05/18）

☁️ 算力服务

5/21 中国移动重庆数智产业创新发展大会：发布三大产业联盟，区域算力生态加速落地

"数联渝州，智赋万物"中国移动重庆公司数智产业创新发展大会成功举行。会上集中展示了通信服务、算力服务、智能服务"三大主业"硬核实力，重磅发布三大产业联盟和合作伙伴十大权益。（来源：新浪新闻 2026/05/22）这是 0501-0510 周报"算力新动能行动计划"在省级落地的具体进展。

🎯 Token 战场

"从卖流量到卖 Token"：三大运营商集体入场，AI 时代新增长引擎雏形显现

新浪财经报道指出，进入 Agent 时代，算力和 Token 消耗成为三大运营商新战场。当传统通信业务的增长引擎接近天花板，Token 这一 AI 时代的最小计价单位，正成为运营商在 AI 时代寻求新增长的关键突围方向之一。（来源：新浪财经 2026/05/22）河北移动等省公司同步推出"Coding Plan"40 元/月套餐，针对学生、AI 从业者，将算力请求与 5G-A 极速网络绑定。

📈 资本市场

资本市场跟进"算力网"叙事：万亿基建逻辑获机构调研关注

中国移动 5月18日披露最新机构调研信息，机构关注点聚焦于"算力新动能行动计划"和全国一体化算力网建设。中国移动 A 股本周一度冲至 103.36 元，5/22 收报 95.98 元，总市值约 2.08 万亿元，仍居通信服务板块首位。（来源：MSN 中文/搜狐财经 2026/05/22）

📋 政策动向

"六张网"路线图浮出水面
🏛️ 发改委

5/22 国家发改委详解"六张网"：年内投资超 7 万亿元，算力网+新一代通信网为核心两环

国家发改委政策研究室副主任、新闻发言人李超在 5月22日发布会上介绍"六张网"的具体内容、投资规模和落实路径——4月28日中央政治局会议明确："水网、新型电网、算力网、新一代通信网、城市地下管网、物流网"，5月9日国务院常务会议进一步部署。今年初步估算"六张网"投资将超 7 万亿元。（来源：新浪财经 2026/05/22）

5月25日新华社进一步透露"六张网"将迎来"时间表""任务书"。其中算力网、新一代通信网两张网直接对应运营商核心业务方向，是本轮基建浪潮中运营商的主要发力点之一。（来源：新华网 2026/05/25）

📜 工信部

工信部第 21 周工作要闻 + APP 通报：监管+治理双线并行

工信部本周发布 2026 年第 3 批（总第 56 批）侵害用户权益 APP/SDK 通报，依据《个人信息保护法》《电信条例》对 31 款产品进行点名。（来源：工信部官方公众号 2026/05）

同期发布"工信领域要闻回顾·2026 年第 21 周"，是部委层面例行的政策传递动作，本周暂无重大新规出台，工作重心在前期已发布的《算力互联互通行动计划》《通信建设工程质量提升和安全生产行动方案》落实推进。（来源：工信部 2026/05）

🌐 6G

中央定调"六张网"，6G 加速从"拼技术"转向"拼价值"

新浪财经 5月13日发布的分析指出，全球 6G 研发已进入关键"窗口期"，叠加中央政治局会议明确加强"六张网"规划建设，6G 标准制定从"技术指标比拼"转向"应用价值比拼"。3GPP 已启动 6G 网络架构、无线空口、安全技术等研究项目，并形成 6G 标准化时间表。（来源：新浪财经 2026/05/13；数字中国建设峰会官网 2026/04/07）
