## 注意事项

### 如果使用Python脚本：

1. **腾讯企业邮箱**：
   - SMTP服务器：`smtp.exmail.qq.com`
   - 端口：465（SSL）
   - 可能需要开启SMTP服务并生成授权码

2. **QQ邮箱**：
   - SMTP服务器：`smtp.qq.com`
   - 端口：465（SSL）
   - 需要开启SMTP服务并使用授权码（不是QQ密码）

3. **163邮箱**：
   - SMTP服务器：`smtp.163.com`
   - 端口：465（SSL）

## 故障排查

### 如果Python脚本报错：

```bash
# 检查是否安装Python
python3 --version

# 脚本报错时查看详细错误信息
python3 send_email.py
```

### 常见错误：

1. **Authentication failed**: 密码或授权码错误
2. **Connection refused**: 检查SMTP服务器和端口
3. **SSL Error**: 尝试使用 `smtplib.SMTP` 替代 `SMTP_SSL`
