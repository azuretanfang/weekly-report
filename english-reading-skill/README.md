# 英文精读报告生成器 📚

一个AI驱动的英文文章深度分析工具，帮助你高效掌握英文精读技巧。

## ✨ 功能特性

### 1️⃣ **智能内容分析**
- 🔍 自动识别常见错误句型（仅英文）
- 📊 生词标注 + 词汇等级分析
- 📈 文章难度等级评估（CEFR: A1-C2）
- 🎯 长难句拆解 + 语法树展示

### 2️⃣ **交互式精读报告**
- 📖 双栏响应式设计（原文 + 精读注解）
- 🎨 生词高亮、难句标注、语法标签
- 🔗 联动滚动（点击注解自动定位到原文对应位置）
- 🔊 IPA 音标 + 例句 TTS 朗读（基于 Google gTTS，可选）

### 3️⃣ **报告导出**
- 💾 HTML 精读报告（浏览器打开即用，可通过浏览器打印保存）

> 🪧 自 v1.2.0 起已移除 Anki 单词卡生成与导出能力，请直接使用 HTML 精读报告进行学习与复习。
>
> 🆕 **v1.3.0（2026-05-07）**：CEFR 等级判定改用 `textstat`（Flesch-Kincaid Grade + Dale-Chall）替代经验公式，修复"任何文本被打成 C2"的算法缺陷；词汇分级新增 `wordfreq` Zipf 频率作为 fallback 主链，advanced 词召回率从 0 提升至 30%+。详见 [`REMEDIATION_NOTES_v1.3.0.md`](./REMEDIATION_NOTES_v1.3.0.md)。

## 📥 输入方式

```
✅ 支持的输入：
- 直接粘贴英文文本
- 输入网页URL（自动抓取正文）
- 上传 .txt 或 .md 文件

⚠️ 目前仅支持英文内容
```

## 🎮 使用步骤

1. **输入内容** → 选择输入方式（文本/URL/文件）
2. **AI分析** → 系统自动识别错误句型、生词、难句
3. **精读学习** → 使用双栏报告进行交互式阅读
4. **保存复习** → 保存 HTML 或通过浏览器打印保存，反复回读

## 🎯 常见错误句型识别

系统会自动识别以下类型的错误：

- ❌ **搭配错误** - 词汇搭配不当
- ❌ **时态误用** - 英文时态使用错误
- ❌ **冠词问题** - a/an/the 使用不当
- ❌ **单复数** - 名词单复数不一致
- ❌ **介词错误** - 介词搭配错误
- ❌ **语序问题** - 语序调整建议

## 📊 词汇等级标准

- 🟢 **Common** - 高频常用词
- 🟡 **Intermediate** - CET4/CET6 等级
- 🔴 **Advanced** - GRE/TOEFL/IELTS 等级

