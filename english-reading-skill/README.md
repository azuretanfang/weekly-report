# 英文精读报告生成器 📚

一个AI驱动的英文文章深度分析工具，帮助你高效掌握英文精读技巧。

## ✨ 功能特性

### 1️⃣ **智能内容分析**
- 🔍 自动识别常见错误句型（仅英文）
- 📊 生词标注 + 词汇等级分析
- 📈 文章难度等级评估（CEFR: A1-C2）
- 🎯 长难句拆解 + 语法树展示

### 2️⃣ **交互式精读报告**
- 📖 双栏响应式设计（原文 + 精读注解）
- 🎨 生词高亮、难句标注、语法标签
- 🔗 联动滚动（点击注解自动定位到原文对应位置）
- 🎤 支持朗读（示例音频）

### 3️⃣ **单词本 + 导出**
- 💾 LocalStorage 单词本管理（收藏/删除）
- 🃏 Anki 导出（`.apkg` 格式）
  - 包含音标（IPA格式）
  - 例句音频（Google TTS）
- 📄 PDF / Markdown 导出选项

## 📥 输入方式

```
✅ 支持的输入：
- 直接粘贴英文文本
- 输入网页URL（自动抓取正文）
- 上传 .txt 或 .md 文件

⚠️ 目前仅支持英文内容
```

## 🎮 使用步骤

1. **输入内容** → 选择输入方式（文本/URL/文件）
2. **AI分析** → 系统自动识别错误句型、生词、难句
3. **精读学习** → 使用双栏报告进行交互式阅读
4. **收集生词** → 点击生词添加到单词本
5. **导出学习** → 生成 Anki 卡片或其他格式

## 🎯 常见错误句型识别

系统会自动识别以下类型的错误：

- ❌ **搭配错误** - 词汇搭配不当
- ❌ **时态误用** - 英文时态使用错误
- ❌ **冠词问题** - a/an/the 使用不当
- ❌ **单复数** - 名词单复数不一致
- ❌ **介词错误** - 介词搭配错误
- ❌ **语序问题** - 语序调整建议

## 📊 词汇等级标准

- 🟢 **Common** - 高频常用词
- 🟡 **Intermediate** - CET4/CET6 等级
- 🔴 **Advanced** - GRE/TOEFL/IELTS 等级

## 💾 Anki 导出格式

导出的卡片包含：

```
正面：生词 + 音标 + 例句
反面：中文释义 + 例句翻译 + 音频
```

例句音频通过 Google TTS 自动生成。
