# 英文精读报告生成器

AI驱动的英文文章精读分析工具。输入英文文本、URL或文件，自动生成包含常见错误句型识别、生词标注、词汇等级分析的交互式精读报告，支持Anki卡片导出（含音标/例句音频）、PDF/Markdown导出。

## 触发场景

当用户提出以下需求时使用此技能：
- 分析/精读英文文章或段落
- 英文生词标注和词汇分析
- 识别英文中的语法错误或常见错误句型
- 生成英文精读报告
- 导出Anki卡片、单词本
- 评估英文文章难度等级（CEFR: A1-C2）
- 英文长难句拆解

## 触发关键词

英文精读、精读报告、英文分析、生词标注、词汇分析、语法识别、Anki导出、英文学习、难度分析、长难句、英文阅读

## 功能

1. **智能内容分析** - 自动识别常见错误句型、生词标注、词汇等级分析、文章难度评估
2. **交互式精读报告** - 双栏响应式设计（原文+精读注解），生词高亮、难句标注
3. **单词本管理** - LocalStorage单词本，支持收藏/删除
4. **多格式导出** - Anki卡片（含音标+TTS音频）、PDF、Markdown

## 输入方式

- 直接粘贴英文文本
- 输入网页URL（自动抓取正文）
- 上传 .txt 或 .md 文件

> ⚠️ 目前仅支持英文内容
