### 卡片 #004：真实性底线 - 表达包装可接受，虚假承诺不可接受

**类别**：价值观·道德边界  
**置信度**：高  
**创建时间**：2026-03-03  
**最后更新**：2026-03-03

#### 核心内容
"弄虚作假是绝对不会妥协的"，但对"真实性"有清晰的边界判断：
- ✅ **表达优化可接受**：把"进度70%"说成"基本完成"视为委婉表达和轻度包装
- ❌ **虚假承诺不可接受**：业务方要求"先说能做，后面再想办法"无法接受，因为"如果后面做不到怎么办"
- ⚠️ **数据美化视情况而定**：看影响大小

**核心边界**：表达优化OK，但虚假承诺NO——因为后者会导致信任崩塌和责任追溯。

#### 证据来源
- 2026-03-03 深度访谈：关于"弄虚作假"的三个场景判断
- 底层逻辑：与"契约精神"一致，核心是"不制造虚假预期"

## 📊 本层总结

### 核心价值观关键词
- **契约精神**：承诺必须兑现，风险必须提前告知
- **责任本位**：本职工作是底线，不接受"摸鱼"
- **行动优先**：拒绝拖延，立即行动
- **真实性底线**：表达可优化，但不可虚假承诺
- **流程依赖**：用规则获得安全感和话语权
- **双轨决策**：工作看组织价值，生活看个人兴趣
- **极强韧性**：持续推动，不轻易放弃

### 内在逻辑
这些价值观形成了一个**自洽的系统**：
1. 相信契约和规则（#001 + #005）
2. 以责任为本，拒绝拖延（#002 + #003）
3. 在真实性上有灵活但有底线（#004）
4. 工作中向上对齐，生活中忠于自己（#006）
5. 遇到困难时持续坚持（#007）

核心驱动力：**责任感 + 规则信仰 + 极强执行力**
