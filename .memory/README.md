# 个人AI记忆系统

## 📖 系统说明

这是一个**持续学习、动态更新的个人记忆系统**，用于帮助AI更好地理解我的价值观、性格特征、心智模式、思维模式、工作习惯、行为偏好、决策原则等，从而实现更高效的协作，甚至模拟我的决策。

---

## 🏗️ 四层金字塔架构

```
       L4 内核层 - 价值观与核心原则
      （极少变动，底层信念）
           ▲
           │
       L3 模式层 - 思维模式与决策框架
      （缓慢迭代，心智模型）
           ▲
           │
       L2 偏好层 - 工作习惯与行为偏好
      （定期更新，表层特征）
           ▲
           │
       L1 事实层 - 具体事件与交互记录
      （持续积累，原始数据）
```

### 各层说明

- **L4 内核层**：价值观、人生信念、核心原则（如契约精神、责任本位、真实性底线）
- **L3 模式层**：思维模式、决策框架、心智模型（如直接沟通、流程依赖、信任评估）
- **L2 偏好层**：工作习惯、沟通风格、行为偏好（如角色定位、情绪反应、风险管理）
- **L1 事实层**：具体事件、项目记录、交互日志（如访谈记录、工作案例）

## 🔄 记忆生命周期

```
采集 → 提炼 → 固化 → 迭代
  ↑                      ↓
  └──────── 验证 ←────────┘
```

1. **采集**：通过访谈、日常交互、工作记录等收集原始信息
2. **提炼**：从具体事件中抽象出模式、原则、偏好
3. **固化**：形成结构化记忆卡片，存入对应层级
4. **迭代**：基于新的信息验证、更新、删除过时记忆
5. **验证**：在实际场景中测试记忆的准确性

## 🎯 使用方式

### 对于AI
- 在与我交互时，参考相应层级的记忆
- 根据任务类型（决策建议/沟通协调/情绪支持）调用不同层级
- 在日常交互中持续学习和迭代记忆

### 对于我自己
- 定期回顾记忆卡片，进行自我认知校准
- 在重要决策时，查看相关的决策框架和价值观
- 记录重要事件和思考，作为L1层原始材料

## 📊 记忆卡片结构

每张记忆卡片包含：
- **标题**：简洁描述（20字以内）
- **层级**：L1/L2/L3/L4
- **类别**：价值观/思维模式/工作习惯/具体事件等
- **内容**：核心描述（100-300字）
- **证据来源**：访谈记录/工作案例/日常交互
- **置信度**：高/中/低
- **创建时间**：YYYY-MM-DD
- **最后更新**：YYYY-MM-DD
- **关联记忆**：相关卡片ID
